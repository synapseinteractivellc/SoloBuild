# 🧙‍♂️ SoloBuild

A browser-based idle RPG where discovery, knowledge, and incremental progress shape your journey.

---

## 📖 Overview

**SoloBuild** is a solo-developed idle/clicker RPG that blends exploration, discovery, and progression systems into a living world. Instead of unlocking features through menus, players uncover new mechanics organically through exploration, interaction, and experimentation.

This project is being actively developed as both a game and a learning experience in system design, programming, and long-term project execution.

---

## 🎯 Core Philosophy

- **Discover → Unlock → Improve → Repeat**
- Systems are unlocked through **exploration and knowledge**, not menus
- The world evolves as the player progresses
- Designed to be **idle-friendly**, with optional interaction for optimization

---

## 🧠 Key Features (Planned & In Progress)

### 🌲 Exploration
- Explore the wilds to uncover resources, events, and hidden locations
- Discover new areas like ruins, dungeons, and ancient structures

### 🏚️ World Discovery
- Find and rebuild locations such as an **Abandoned Tower**
- Unlock new systems by interacting with the world

### 🧪 Crafting & Knowledge
- Learn through **books, tomes, and study systems**
- Craft items like healing salves and potions
- Knowledge unlocks new capabilities rather than raw power

### ⚔️ Combat & Dungeons
- Engage in automated combat with optional player interaction
- Discover and clear multi-stage dungeons
- Earn rare drops, resources, and knowledge items

### 🧑‍🌾 NPC Interactions
- Build relationships with characters like the Herbalist
- Unlock new actions and systems through dialogue

### ⚙️ Idle Progression
- Perform actions over time
- Future plans include automation and worker systems

---

## 🛠️ Tech Stack

- **JavaScript (Vanilla)**
- HTML / CSS
- Modular file structure:
  - `state.js` — game state management
  - `actions.js` — player actions
  - `combat.js` — combat systems
  - `render.js` — UI rendering
  - `save.js` — save/load system
  - `main.js` — utilities
  - `bootstrap.js` — initialization
